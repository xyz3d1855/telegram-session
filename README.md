# Telegram Account Manager Pro

这是一个基于 `Telethon` 库开发的 Telegram 账号多功能管理工具。支持账号状态批量检测、历史消息导出以及特定目标的实时消息监听。

> **Assistant's Note**: 本项目由 Trae AI 协助开发。代码经过脱敏处理，去除了所有硬编码的个人信息与凭据，采用配置驱动设计，适合作为开源模板或个人管理工具使用。

---

## 🌟 主要功能

- **多账号管理**：支持添加多个账号，并以 `StringSession` 格式持久化存储，无需担心 `.session` 文件丢失。
- **状态检测**：一键批量检测所有账号是否被封禁（Ban）或受限（Limited）。
- **历史记录导出**：全量导出与指定目标的聊天记录（支持 777000 等服务号）。
- **实时监听**：实时监控指定目标的动态，并将新消息自动同步到本地日志。
- **全异步设计**：基于 `asyncio`，高效处理网络请求与消息监听。

## 🚀 快速开始

### 1. 环境准备
确保你的系统已安装 Python 3.8+。安装依赖：
```bash
pip install telethon cryptg
```

### 2. 配置凭据
1. 复制示例配置文件：
   ```bash
   cp config.json.example config.json
   ```
2. 编辑 `config.json`，填入你的 `api_id` 和 `api_hash`（从 [my.telegram.org](https://my.telegram.org) 获取）。
3. 根据需要配置代理（支持 SOCKS5）。

### 3. 运行程序
```bash
python tg_manager_template.py
```

## 📖 操作指南

- **添加账号 (A)**：按照提示输入手机号及验证码。
- **全体检测 (C)**：程序会模拟与 `@SpamBot` 对话来判断账号当前的健康状态。
- **导出历史 (E)**：选择一个正常账号，输入目标 ID，程序将导出完整的历史对话。
- **实时监听 (L)**：选择一个账号开启监听模式，所有新消息将实时打印并保存。

## ⚠️ 注意事项

1. **隐私安全**：请勿将 `config.json` 或 `sessions/` 目录上传到任何公共仓库（如 GitHub）。本项目已默认在配置中建议忽略这些文件。
2. **频率限制 (Flood)**：频繁检测或快速翻页导出历史记录可能会触发 Telegram 的频率限制（FloodWait）。脚本已内置了一定的延迟，但仍建议不要在短时间内操作过多账号。
3. **风控风险**：使用自动化工具登录账号存在一定的风控风险，建议配合稳定的代理使用。
4. **免责声明**：本项目仅供学习与技术交流使用，请遵守 Telegram 的服务条款。因不当使用导致的账号封禁等后果由使用者自行承担。
